## Home Assistant Automation

This is an agent skill for AnythingLLM that allows you to automate your Home Assistant devices.

## Setup

1. Install the package by using its import ID from the AnythingLLM Community Hub and pasting that into the "Import ID" field in the AnythingLLM app.
2. Set up your Home Assistant API URL and API key in the AnythingLLM app into the "Home Assistant API URL" and "Home Assistant API Key" fields.
3. Activate the skill by clicking the toggle button in the AnythingLLM app under the "Agent Skills" section.

## Usage

Once installed, you can ask the AnythingLLM app via `@agent` to automate your Home Assistant devices using natural language.

```
@agent turn on the living room lights
```
